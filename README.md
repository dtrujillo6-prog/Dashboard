#### Video Demo: <URL HERE>
#### Description:
My Photoshoot Planner is a project inspired by my professional need to constantly build mood boards for photoshoots.
The idea here is to take a tool that I use frequently like Milanote and remove all the bloat tools that I dont need to do my job.
I wanted to create a simplified board that only has the tools that I use. In doing this I have created a space where images can be uploaded, color coded, and labeled. 
Once the board has been built you can then write specific notes about the project on a separate notepad and build a photoshoot checklist (which I love).
There is also an integrated google calendar so you can schedule photoshoots.
After you have filled out everything necessary for the projects conclusion you can export the project/save as pdf or png/ or email/airdrop the project right from the app itself.

Any design decisions you debated — explain the choice you made
I actually did debate my design layout and completely changed it! When I started building this app I told AntiGravity what I wanted to build and let the app make all my design decissions for me.
While it did technically give a working project, the UX was terrible, the functionality wasnt quite there, and the way it was styled was so basic and not what I would want for myself. 
So I plugged in Codex to my repo and decided to start being really specific with what I wanted the design and UX to be. I found that codex took my instruction VERY well and was able to impliment my changes with little to no hiccups. 
The end result is a clean and sleek looking photoshoot planner thats styled in black,white,grey, with accents of chrimson red. The UX still needs some work, however its completely functional and I will likely keep working on the UX.
